## Usage

### Default
<Preview>
<Story name='Default'>
{() => '<fw-input label="Name" placeholder="Enter your first name"></fw-input>'}
</Story>
</Preview>

### Required
<Preview>
<Story name='Required'>
{() => '<fw-input label="Name" required="true" placeholder="Enter your first name"></fw-input>'}
</Story>
</Preview>

### Disabled
<Preview>
<Story name='Disabled'>
{() => '<fw-input label="Name" required="true" disabled="disabled" placeholder="Enter your first name"></fw-input>'}
</Story>
</Preview>

### With State text
<Preview>
<Story name='StateText'>
{() => '<fw-input label="Name" state-text="It has to be your name" required="true"  placeholder="Enter your first name"></fw-input>'}
</Story>
</Preview>
